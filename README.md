# AI POWERED DATA ANALYSIS REMOTE INTERNSHIP

An automated analytics pipeline built in Python to ingest, process, and summarize global student participation data. This project analyzes a dataset containing over 8,500 registration entries to identify baseline student conversion patterns across various educational categories and global geographic landscapes.

## 👥 Team Assignments & Roles
* **Member 1 (Data Analyst):** Handled quantitative aggregations, metrics extraction, and text reporting structures.
* **Member 2 (Data Visualizer):** Developed visual scripts for categorical distributions and plotting demographic funnels.

---

## 📊 Core Baseline Statistics
Our initial data overview surfaced the following high-level participation trends:
* **Total Clean Observations:** 8,558 student rows.
* **Dominant Operational Tracks:** Experiential *Internships* account for **63.34%** of overall engagement volume, followed by specialized *Courses* at **23.80%**.
* **Geographic Concentration:** Activity is heavily centered in the **United States (46.46%)** and **India (33.14%)**.
